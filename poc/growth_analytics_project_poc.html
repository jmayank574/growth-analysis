experiment design layer
We treat the 8 unique landing pages (landing_page_id) as a natural A/B test. Using chi-square significance testing in Python we determine which landing pages statistically outperform others on conversion rate — without needing to run a new experiment.
tools
SQL CTEs + window functions pandas pivot_table scipy.stats (chi-square, t-test) plotly funnel charts
